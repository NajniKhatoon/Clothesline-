# Ultras – Casual Wear Store (Clothesline)

A responsive, static e-commerce landing page for a fictional casual-wear clothing brand called **Ultras**. Built with plain HTML and CSS, it showcases a full front-page layout including a hero banner, featured products, best-selling items, flash sales, and a footer.

## 🖥️ Live Preview

Open `home.html` in your browser to view the site locally.

## 📁 Project Structure

```
myweb/
├── home.html                  # Main homepage markup
├── home.css                   # All styling for the homepage
├── *.jpg / *.webp              # Product images used throughout the page
└── README.md                  # Project documentation
```

## ✨ Features

- **Top announcement bar** – contact number, shipping promo, and quick-access icons (profile, cart, wishlist, search)
- **Navigation bar** – brand logo and primary site navigation
- **Hero section** – large banner promoting the current "Casual Collection"
- **Featured products grid** – 3-column product showcase with hover overlay (Add to Cart, Quick View, Wishlist)
- **Split hero banners** – promotional tiles for Street Wear, Basic Shoes, and Woolen Hats
- **Best-selling products** – filterable category tabs (Shoes, T-shirts, Pants, Hoodies, etc.) with a 4-column product grid
- **Flash sales section** – discounted items with strike-through pricing and sale badges
- **Quote of the day** – styled blockquote section
- **Trust bar** – shipping, returns, and support highlights with inline SVG icons
- **Footer** – site links, contact info, social icons, and payment method logos

## 🛠️ Built With

- **HTML5** – semantic page structure
- **CSS3** – Flexbox & CSS Grid for layout, hover transitions, and responsive design
- No frameworks, build tools, or external JS libraries — pure HTML/CSS

## 🚀 Getting Started

### Prerequisites
Just a modern web browser (Chrome, Firefox, Safari, Edge).

### Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/NajniKhatoon/Clothesline-.git
   ```
2. Navigate into the project folder:
   ```bash
   cd Clothesline-
   ```
3. Open `home.html` directly in your browser, or use a local dev server (e.g. VS Code Live Server extension) for the best experience.

## 📌 Notes

- All product images are stored locally in the project root and referenced with relative paths.
- Payment icons (Visa, Mastercard, Amex) in the footer are loaded from Wikimedia Commons.
- This is a front-end template only — there is no backend, cart logic, or payment processing implemented yet.
